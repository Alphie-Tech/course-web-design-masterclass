# 🚀 VS Code Setup Guide – Alphie Tech Responsive Frontend Engineering Masterclass (10 Weeks)

Welcome to your development environment setup guide.

Visual Studio Code (VS Code) is the primary code editor we will use throughout this course. It is lightweight, powerful, beginner-friendly, and widely used by professional developers worldwide.

---

# 📥 Step 1 — Install VS Code

## Download VS Code

Visit:

https://code.visualstudio.com/

Download the version for your operating system:
- Windows
- macOS
- Linux

Install normally.

---

# 📦 Step 2 — Install Important Extensions

Open VS Code.

On the left sidebar:
- Click the Extensions icon
OR
- Press:

```bash
Ctrl + Shift + X
```

Install the following extensions:

---

## ✅ Live Server

Extension Name:
```txt
Live Server
```

Author:
```txt
Ritwick Dey
```

Purpose:
- Runs your HTML projects in a live browser
- Automatically refreshes when files change

---

## ✅ Prettier

Extension Name:
```txt
Prettier - Code formatter
```

Purpose:
- Automatically formats your code professionally

---

## ✅ Tailwind CSS IntelliSense

Extension Name:
```txt
Tailwind CSS IntelliSense
```

Purpose:
- Gives Tailwind class suggestions and autocomplete

---

## ✅ ES7+ React/Redux Snippets

Extension Name:
```txt
ES7+ React/Redux/React-Native snippets
```

Purpose:
- Helps write React code faster

---

## ✅ Auto Rename Tag

Extension Name:
```txt
Auto Rename Tag
```

Purpose:
- Automatically renames matching HTML tags

---

## ✅ Path Intellisense

Extension Name:
```txt
Path Intellisense
```

Purpose:
- Helps autocomplete file paths

---

## ✅ Error Lens

Extension Name:
```txt
Error Lens
```

Purpose:
- Displays errors directly inside your code editor

---

# ⚙️ Step 3 — Enable Auto Save

In VS Code:

Go to:
```txt
File → Auto Save
```

Enable it.

This helps save files automatically while coding.

---

# 🎨 Step 4 — Change Theme (Optional)

Recommended themes:
- One Dark Pro
- Tokyo Night
- GitHub Dark

You can install them from Extensions.

---

# 🔤 Step 5 — Increase Font Size (Optional)

Go to:

```txt
File → Preferences → Settings
```

Search:
```txt
Font Size
```

Recommended:
```txt
16
```

Create a folder on your computer:

Example:

```txt
Alphie-Tech-Masterclass
```

- Store all assignments

# 🌐 Step 7 — Install Git

Git is required for GitHub submissions.

Download Git:

https://git-scm.com/downloads

Install normally.

---

# ✅ Verify Git Installation

Open terminal and run:

```bash
git --version
```

You should see something like:

```bash
git version 2.x.x
```
